# Otaku Emporium

An ecommerce website that sells anime products and apparels.

### Project Status
This project is currently in development. As per request, user can sent User Authentication Request through backend. 


### Installation and Setup Instructions
Clone down this repository. You will need node and npm installed globally on your machine.

#### 1. Installation:
npm install

#### 2. To Run Test Suite:
npm test

#### 3. To Start Server:
npm run server

#### 4. To Start Client:
npm run client

#### 5. To Start Server and Client at the same time:
npm run dev

#### 6. To Visit App:
localhost:3000 (Client)

#### 7. To Sent Backend Request 
localhost:500/{request URL}

#### 7. To Run Sass server 
sass --watch ./client/src/stylesheets/index.scss ./client/src/stylesheets/css/index.min.css
